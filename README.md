# Práticas AED — Algoritmos e Estruturas de Dados

Repositório para postagens das listas de algoritmos e estruturas de dados e um pouco do meu aprendizado na matéria.

## Sobre

Este repositório reúne exercícios, implementações e anotações desenvolvidos ao longo da disciplina de **Algoritmos e Estruturas de Dados (AED)**. O objetivo é registrar o progresso e facilitar revisões futuras.

## Estrutura

```
praticas-aed/
├── listas/          # Listas de exercícios propostas na disciplina
├── algoritmos/      # Implementações de algoritmos (busca, ordenação, etc.)
└── estruturas/      # Estruturas de dados (listas, filas, pilhas, árvores, etc.)
```

## Tecnologia

As implementações são feitas principalmente em **C/C++**.

## Como compilar

```bash
gcc -o programa arquivo.c
```
ou
```bash
g++ -o programa arquivo.cpp
```
